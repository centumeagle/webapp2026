# ⚛️ 웹프로그래밍응용 (Web Programming Application)

Webapp 2026
> HTML/CSS 기초를 복습하고 리액트(React)의 핵심 동작 원리와 컴포넌트 기반 설계를 학습합니다.
> 상태 관리, 훅(Hooks), 전역 컨텍스트 등 효율적인 웹 애플리케이션 구축을 위한 실무 기술을 습득합니다.

---

### 🌐 Foundation: HTML & CSS
* **HTML/CSS 복습**: 웹 표준을 준수하는 시맨틱 마크업과 모던 레이아웃 구성 방식 재정립

### ⚙️ React Essentials
* **React CRA**: Create React App을 활용한 최적화된 React 개발 환경 구축 및 보일러플레이트 분석
* **React Element**: 가상 DOM(Virtual DOM)의 개념과 JSX를 이용한 엘리먼트 렌더링 원리
* **React Component**: 데이터 중심의 컴포넌트 설계 및 Props를 통한 컴포넌트 간 정보 전달

### 🔄 Interactive Logic
* **State & Lifecycle**: 컴포넌트의 유기적인 상태 관리 및 렌더링 주기(Lifecycle)의 이해
* **Event Handler**: 사용자 인터렉션 처리를 위한 이벤트 바인딩 및 함수 최적화
* **Conditional Rendering**: 논리 연산자와 삼항 연산자를 활용한 동적 UI 분기 처리
* **Listen Keys**: 유동적인 리스트 데이터 렌더링 시 고유 Key값을 활용한 렌더링 효율 극대화

### 🚀 Advanced React Patterns
* **React Hook**: `useState`, `useEffect` 등 Hook을 활용한 함수형 컴포넌트의 상태 로직 구현
* **Lifting State Up**: 상태 끌어올리기를 통한 인접 컴포넌트 간 데이터 동기화 및 단방향 데이터 흐름 제어
* **React Context**: Context API를 활용한 Prop Drilling 문제 해결 및 전역 상태 관리 체계 구축
* **React Styling**: CSS Modules, Styled-components 등 컴포넌트 기반 스타일링(CSS-in-JS) 기법 적용
